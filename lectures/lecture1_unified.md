**Q7.** Distributed Computing is best defined as:

**Q39.** In the November 2025 Cloudflare outage, what was the root cause?
- A) A DynamoDB failure
- B) A single config file exceeding its limit, causing bot mitigation to fail and BGP routing to break
- C) A natural disaster destroying data centers
- D) An IAM misconfiguration


**Answer:** **B** — Single config file exceeded limit → BGP routing protocol failure

---

**Q40.** In the October 2025 AWS DynamoDB outage, the root cause was:
- A) A hardware failure in EC2 instances
- B) A config file size limit exceeded
- C) An internal DNS race condition where clean-up deleted old records just applied by a slow DNS enactor
- D) A Lambda function timeout

**Answer:** **C** — Internal DNS race condition (clean-up deleted records just applied by slow DNS enactor)
**Topic:** Cascading Failures
